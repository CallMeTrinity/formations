# Consignes de conception des formations

Ces fichiers définissent comment concevoir et rédiger une formation dans ce dépôt. Ils sont la
référence : toute création ou modification de formation doit s'y conformer.

## Ordre de lecture

1. [`charte-pedagogique.md`](charte-pedagogique.md) — **pourquoi et pour qui** : public cible,
   objectif de sortie, principes pédagogiques, ton.
2. [`structure-formation.md`](structure-formation.md) — **comment c'est rangé** : arborescence d'une
   formation, nommage, navigation, statuts.
3. [`format-chapitre.md`](format-chapitre.md) — **à quoi ressemble un chapitre** : anatomie d'un
   fichier, sections obligatoires, encarts, exercices, quiz.
4. [`style-redactionnel.md`](style-redactionnel.md) — **comment on écrit** : langue, voix,
   conventions de code et de markdown.
5. [`workflow-creation.md`](workflow-creation.md) — **comment on procède** : process pas à pas pour
   produire une formation complète.
