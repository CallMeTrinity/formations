All notable changes to this project will be documented in this file.

## [Unreleased]

## [1.1.0] - 2026-06-19

### Added

- Tag management to admin panel and preferences, including search, selection, creation and deletion functionality

## [1.0.0] - 2026-06-19

### Added

- Formation Management: Added an admin panel for managing formations, including creation, enrollment and completion statistics, and a catalogue page with filters for themes and difficulty.
- User Progress Tracking: Implemented a user dashboard for tracking formation progress, with completion history and the ability to mark chapters as completed.
- Recommendation System: Introduced a personalized recommendation system for formations.
- User Authentication and Profile: Added user registration and authentication, profile management (display name, password change), and user preferences.
- UI Enhancements: Redesigned the UI with responsive design, dark mode, mobile optimizations, and a logo replacement.
- Markdown Content Parsing: Improved parsing of formation Markdown content to structure information (prerequisites, objectives) and integrated GithubFlavoredMarkdown conversion.
- Core Functionality: Includes management of content, progression, and user entities, toast notifications, and UI improvements.
